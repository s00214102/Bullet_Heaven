Bullet Heaven

About this project

This game was developed by myself and Jack Laird in 1st year for the Games Programming final project of Computing in Games Development at ATU Sligo, formerly IT Sligo. It was developed in the Unity game engine and written in C#. The theme of the game was 'escape'.

Gameplay

The gameplay revolves around an experimental ball of energy created in a lab, that can harness the power of the elements. However it unexpectedly gains conciousness and breaks free, now you must control its power to escape to the surface.

Elements

There are four elements at your disposal, each with its own unique abilities. You can switch elements by pressing 1,2,3 or 4 for the corresponding element. After switching the element you were previously on will be unavailable for a short duration.

Fire

The fire element specializes in hitting multiple enemies at once, and can set things on fire dealing damage over time.

Ice

The ice element has a very fast fire rate and is useful for slowing down enemy movement and bullet speed.

Earth

The earth element is good at breaking through enemy defences and can destroy their armor. Attacks can stun enemies and it can also create a barrier to take cover behind.

Lightning

The lightning element has high damage and high range, it can also apply shock which increases damage dealt to enemies.

Saving

The main menu featured a save system with one save slot available which tracks your progress through the game as well as items collected.

The player can save their progress by using a checkpoint in a level.
